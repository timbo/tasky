# Tasky API

API for the Tasky task management app

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

When running locally, the API is available at localhost:3000/tasks

## Run tests

```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```
